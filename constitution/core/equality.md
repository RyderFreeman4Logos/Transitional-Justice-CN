# 平等权

法律面前人人平等。

任何人不得因性别、民族、种族、语言、宗教信仰、政治见解、社会出身、财产状况、出生地、身体状况或其他身份而受到歧视。

国家有义务采取措施消除事实上的不平等，促进机会平等。
