# 总统制

总统由全体公民直接选举产生，兼任国家元首和政府首脑。

**行政权：**
- 总统独立行使行政权，组建内阁，任命各部部长
- 内阁成员对总统负责，而非对议会负责
- 总统任期固定（四年或五年），非经弹劾程序不得被免职

**与立法机关的关系：**
- 总统与议会各自独立，互不隶属
- 总统不得解散议会，议会不得以不信任案迫使总统辞职
- 总统对议会通过的法案享有否决权，议会可以超级多数（如三分之二）推翻否决
- 总统不得兼任议员

**制衡机制：**
- 重要人事任命（最高法院法官、内阁部长等）须经议会确认
- 议会掌握预算审批权
- 弹劾程序：议会可在总统犯有严重违法行为时启动弹劾

**优势：** 权力分立明确，行政稳定，选民直接选择行政首长。

**劣势：** 总统与议会分属不同政党时容易陷入僵局；总统权力集中，有滑向强人政治的风险；败选方在任期内无法更换行政首长。
