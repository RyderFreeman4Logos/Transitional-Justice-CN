# 半总统制

总统由人民直接选举产生，掌握部分行政权（主要是国防和外交）。总理由议会多数产生，主持日常政务。

**总统的权力：**
- 由全体公民直选产生，任期固定
- 负责国防、外交和国家安全事务
- 任命总理（但须获得议会信任）
- 可在特定条件下解散议会
- 主持国家安全会议

**总理的权力：**
- 由总统任命，须获得议会多数信任
- 主持内阁会议，负责内政、经济、社会等日常政务
- 对议会负责，议会可通过不信任案迫使总理辞职

**共治（Cohabitation）：**
- 当总统与议会多数分属不同政党时，总统须任命对方阵营的总理
- 此时行政权实际向总理倾斜，总统退居国防外交角色

**优势：** 总统直选赋予强民主正当性；总理对议会负责避免僵局；权力在两个行政首长之间分散，降低独裁风险。

**劣势：** 总统与总理之间可能产生权力冲突；权责边界模糊；共治时期决策效率可能降低。
