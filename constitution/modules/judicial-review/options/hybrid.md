# 混合制

设立宪法法院行使抽象审查和机关争议裁决权，同时保留普通法院在个案中审查法律合宪性的权力。

**宪法法院管辖：**
- 抽象审查（特定机关提起）
- 机关争议
- 政党违宪审查

**普通法院管辖：**
- 个案中的合宪性审查（附带审查）
- 如普通法院认为某项法律违宪，可直接在个案中拒绝适用
- 对涉及根本性宪法问题的案件，普通法院可（但非必须）提交宪法法院裁决

**冲突解决：** 宪法法院的抽象审查裁判优先于普通法院的个案判断。普通法院对法律合宪性的判断如与宪法法院此前的裁判相矛盾，以宪法法院为准。

本选项兼顾了集中审查的权威性和分散审查的可及性。
