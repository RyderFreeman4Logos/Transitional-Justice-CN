# 单一制（集权型）

国家为单一制共和国。全国实行统一的法律、行政和司法制度。

中央政府统一行使国家权力。地方政府为中央政府的派出机构，在中央授权范围内执行政务。

各省设立民选议会，但行政首长由中央政府任命（或由民选议会提名，中央政府确认）。

全国实行统一的财税制度，由中央统一征收后向地方转移支付。

本选项适用于转型初期维持国家统一和行政效率的需要。宪法可规定在过渡期结束后，通过修宪程序逐步扩大地方自治权。
