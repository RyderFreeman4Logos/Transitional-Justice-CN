# 单一制（分权型）

国家为单一制共和国。地方行政区享有法律保障的自治权。

中央政府负责全国性事务。省级政府在法律授权范围内享有自主管理本地区事务的权力，包括：
- 地方财政与税收征管
- 教育与公共服务的具体实施
- 城市规划与地方基础设施
- 地方性法规的制定（不得与国家法律抵触）

各省设立民选议会和民选行政首长。中央不得任意撤换地方民选官员。

少数民族聚居地区可依法设立自治区域，在语言、文化、教育等方面享有额外的自治权利。

中央对地方的监督以合法性审查为限，不得干预地方在自治权范围内的决策。
