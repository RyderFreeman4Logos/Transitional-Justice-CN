# 联邦制

国家为联邦共和国，由各联邦成员单位（省、自治区或邦）组成。

**中央与地方权力划分：**

联邦专属事务：
- 国防与外交
- 货币与中央银行
- 关税与对外贸易
- 联邦司法体系
- 移民与国籍

地方专属事务：
- 教育与文化
- 地方警察与公共安全
- 城市规划与土地利用
- 地方交通与基础设施

共享事务（联邦立法，地方执行）：
- 税收（联邦税与地方税并存）
- 环境保护
- 社会保障
- 公共卫生

各联邦成员单位制定本地区宪法，选举本地区议会和政府。联邦成员单位的宪法不得违反联邦宪法的基本原则。

设立联邦参议院，由各联邦成员单位的代表组成，参与联邦立法。
