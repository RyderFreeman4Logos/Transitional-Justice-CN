# 仅允许被动安乐死

允许放弃治疗和拒绝延命措施，但禁止任何主动致死行为。

**合法行为：**
- 患者有权拒绝任何医疗措施，包括维持生命的治疗
- 患者有权要求停止已开始的延命治疗
- 医生可实施缓和医疗（palliative care），即使客观上可能缩短生命
- 事前指示（生前预嘱）具有法律效力

**非法行为：**
- 任何人不得主动采取致死行为
- 医生不得开具或提供致死药物

国家应大力发展缓和医疗和临终关怀服务，确保临终患者获得有尊严的照护。
