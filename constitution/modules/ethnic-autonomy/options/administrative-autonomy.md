# 行政自治

自治区享有较大的行政管理权限，在文化自治基础上增加行政自主权。

**自治权限：**
- 文化自治的全部权利
- 自治区政府对地方事务享有行政管理权
- 地方财政收入的一定比例留归自治区使用
- 自然资源开发收益的合理分成
- 自治区有权制定地方性行政法规（不得与国家法律抵触）
- 自治区有权根据本地情况调整中央政策的执行方式

**人事安排：**
- 自治区行政首长由本地区民主选举产生
- 自治区议会由本地区选民选举产生

**中央保留权力：**
- 国防、外交
- 货币、关税
- 刑法和基本民事法律
- 宪法解释和违宪审查
