# 自决权

特定地区的人民有权通过民主程序决定自身的政治地位，包括自治程度和是否独立。

**适用条件：**
- 适用于具有独特历史、文化和民族认同的地区
- 须经过充分的社会讨论和谈判期（不少于五年）
- 公投前须由独立委员会确认当地居民的真实意愿不受胁迫

**公投选项：**
公投至少应包含以下选项：
1. 维持现有自治安排
2. 扩大自治权（高度自治）
3. 成为联邦成员邦（完全平等地位）
4. 独立

**公投程序：**
- 须有不低于该地区合格选民 60% 的投票率
- 独立选项须获得有效票的超级多数（如 60%）方可通过
- 公投结果须经宪法法院确认程序合法

**独立后安排：**
- 若公投通过独立，须经不少于两年的过渡期进行谈判
- 谈判内容包括：边界划定、国籍选择、财产处理、债务分担、少数民族保护
- 新独立国家对其境内的少数民族负有与本宪法核心条款同等的人权保障义务
