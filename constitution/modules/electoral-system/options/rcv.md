# 排序复选制 (RCV)

选民按偏好顺序对候选人排序（第一选择、第二选择……）。计票时，若无人获得过半数第一选择票，淘汰得票最低的候选人，其选票按第二选择重新���配。重复此过程直到某候选人获得过半数。

**优势：**
- 当选者获得多数支持（至少是转移后的多数），消除"少数票当选"问题
- 选民可以真实表达偏好，无须策略性投票
- 有利于温和派候选人（需要争取对手支持者的第二选择）
- 减少负面竞选（攻击对手可能失去其支持者的第二选择）
- 保留地方代表性（仍然是单选区制）

**劣势：**
- 计票规则比 FPTP 复杂，需要选民教育
- 仍然���单席位选区，小党获得议席的难度仍然较高（优于 FPTP 但不如比例代表制）
- 排序过程可能出现非单调性悖论（提高排名反而落选的极端情况）
- 需要集中计票（不能在投票站层级逐步汇总）

**变体：** 多议席选区下的排序复选制即为"可转移单票制"（STV），兼具比例代表性，澳大利亚参议院和爱尔兰采用。
