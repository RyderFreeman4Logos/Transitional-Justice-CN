# 比例代表制

议席按各政党的全国（或大选区）得票比例分配。选民投票给政党名单而非个人候选人。

设置最低得票门槛（如 3%-5%），未达门槛的政党不能获得议席。

**优势：**
- 议席分配最接近民意分布，代表性最强
- 少数派和新兴政治力量有公平的获得议席机会
- 几乎不存在"浪费选票"的问题
- 无选区划分操纵问题

**劣势：**
- 容易造成政党碎片化，需要多党联合组阁
- 联合政府可能不稳定，小党拥有超出其体量的议价权
- 选民与具体代表之间缺乏地理联系
- 政党内部的名单排序可能不够透明

**变体：**
- 封闭名单制：选民只能投政党，名单排序由党内决定
- 开放名单制：选民可以在政党名单内选择特定候选人，影响当选顺序
