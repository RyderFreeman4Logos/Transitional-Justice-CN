# 联立制 (MMP)

每位选民投两票：一票投给本选区候选人（选区票），一票投给政党（政党票）。

议席分两部分：
- **选区议席**（约一半）：各选区得票最多者当选（类似 FPTP）
- **政党名单议席**（约一半）：用于补偿，使各党总议席数与政党票得票比例一致

设置最低门槛（如得票 5% 或赢得 3 个选区议席）。

**优势：**
- 兼顾地方代表性（每个选区有自己的代表）和全国比例性
- 最终议席比例高度接近各党得票比例
- 被广泛认为是"最平衡"的选举制度设计
- 选民可以分裂投票（选区投人、政党票投党）

**劣势：**
- 规则较复杂，需要充分的选民教育
- 产生两类议员（选区议员 vs 名单议员），可能造成权责不清
- 超额议席（Overhang seats）问题：当某党选区议席超过其比例份额时的处理
- 仍然受选区划分影响（选区票部分）

**注意区分"联立制"与"并立制"：** 日本和台湾采用的是并立制（parallel system），选区票和政党票独立计算，不做补偿，因此比例性远不如 MMP。
