# 单选区简单多数制 (FPTP)

全国划分为若干单一议席选区。每个选区中得票最多的候选人当选，无须获得过半数选票。

**优势：**
- 规则极其简单，选民易于理解
- 每个选区有一名明确的代表，地方代表性强
- 倾向产生两大党轮替的稳定格局，有利于组建多数政府

**劣势：**
- 大量选票被"浪费"（落败候选人的选票和超过当选门槛的多余选票）
- 一个政党可以凭不到半数的全国选票获得多数议席
- 小党和新兴政治力量极难获得议席
- 鼓励策略性投票（"含泪投票"）而非真实偏好表达
- 选区划分（gerrymandering）可严重扭曲选举结果
